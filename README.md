# Dotfiles
Personal dotfiles for macOS.

## Prerequirements
- [Homebrew](https://brew.sh/)

## Usage
```bash
git clone https://github.com/ttksm/dotfiles.git
cd dotfiles
./install.sh
```

## Support programming language
- Go
  - goenv (anyenv)
- Node.js
  - nodebrew (Homebrew)
- Python
  - pyenv (Homebrew)
- Ruby
  - rbenv (Homebrew)
- JVM
  - SDKMAN (Native)
